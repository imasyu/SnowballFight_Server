<仕様書>----------------------------------------------------
ゲーム内容
プレイヤーは雪だるまになり、雪玉を転がして敵をノックバックさせステージ外に弾き出す。

クライアント
Mode1を選択。
サーバのIpv4アドレスをPortファイルに入力し起動。

サーバー
Mode0を選択し起動

※同じLAN内での通信のみ対応。なお、VPNを構築することでリモートでの通信も可能。

***IPとサーバクライアントの変更方法***
アセット→ゲームセッティング内のメモで変更する

<操作方法>
移動:WASD
雪玉射出:SPACE
